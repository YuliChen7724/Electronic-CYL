# Data
This folder stores text entries generated by the emotion module (Electronic CYL).

Each entry represents a single emotional impulse or desire-driven thought,
recorded for the purpose of structured reflection and later NLP exploration.

At this stage, entries are written manually and may be annotated with simple tags.

#此文件夹用于存放由情绪模块（Electronic CYL）生成的文本条目。每一条记录代表一次独立的情绪冲动或由欲望驱动的想法，其目的在于支持结构化反思以及后续的 NLP 探索。在当前阶段，这些条目以人工方式记录，并可能附带简单的标签注释。
